# 《龙场OS》

以阳明心学为 ground truth 的 agentic operating system 研究书稿。

这不是一份零散笔记，也不是若干 prompts 的合集。它试图把“心即理、致良知、知行合一、事上磨炼、破心中贼、传习”等阳明学纲领，压入现代 agentic systems 的原则层、省察层、执行层、记忆层、控制面与安全结构之中，写成一部既可阅读、又可实现、还可继续扩写的系统之书。

## 开始阅读

- [从《序》开始](./00-序.md)
- [先看《执行摘要》](./01-执行摘要.md)

## 阅读说明

- 左侧侧边栏按照全书顺序排列，可依次通读。
- 每一卷都尽量同时回答四件事：概念是什么、系统映射是什么、失控模式是什么、工程落点是什么。
- 当前版本已可连续阅读，但仍是持续扩写中的研究母稿。
