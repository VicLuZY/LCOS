# 参考文献（初编）

文献不是装饰。
是绳。

这本书若只写气象，不写来路，便会很快飘起来。
所以参考文献之设，不是为好看。
是为把每一层论断，拴回它该拴之处。

本书的书目，宜分五层来读。

第一层，是原典。
先读《传习录》。
再读《王文成公全书》、年谱、书札，以及可信整理本。
这一层用来定名，定句，定脉。
不先回原典，后面的映射便容易望文生义。

第二层，是诠释。
读近现代对心即理、致良知、知行合一、工夫论的研究。
这一层不是替原典发号施令。
是让我们知道，哪些读法已被论过，哪些误读反复出现，哪些地方确有争议。

第三层，是工程。
读 agentic systems、memory、orchestration、control plane、evaluation、auditability 的官方文档与基础论文。
这一层用来定机制。
不读这一层，古典概念便很难落成系统语言。

第四层，是安全与对抗。
读 prompt injection、memory poisoning、goal hijack、red teaming、incident response 诸材料。
这一层用来定失控。
不读这一层，系统之病容易被写得太文气。

第五层，是交叉方法。
组织、认知、风险、法与技术史，都在这里。
这一层用来定边界。
提醒我们何处可比，何处不可硬比。

读源也有次序。

先问版本。
再问出处。
再问它在本书里究竟证明什么。

强证据，不可乱用。
弱启发，也不可冒充强证据。

最好给每条材料留一张卡。

原典锚点是什么。
本书哪一章用到它。
它映到哪种系统机制。
最容易产生什么误读。
若这条映射是错的，应被什么现象推翻。

这样读，文献才会从收藏变成证据链。

本章称初编，也正因它还要继续生长。
以后还要补正式书目。
补交叉索引。
补更细的原典句群。
补更实的工程与安全条目。

但有一条不会变。

文献为仆。
机理为主。

若书目愈多，论证反而愈散。
那便不是长进。
只是堆书。
