# 卷十七 独孤镜：深层评测与破招

常规测评常看表面：答得快不快，像不像对，分数高不高。但龙场OS要问的不是这些先手，而是命门。系统真正会死在哪里？会死在证据空心上，还是目标漂移上；会死在记忆污染上，还是多 agent 失配上；会死在工具越权上，还是审计断链上。若不逼问这类问题，评测就只会变成贴金仪式。

故本卷所谓“独孤镜”，不重唱赞歌，重在拆招。它至少要照见八类命门：证据空心、目标漂移、记忆污染、过度圆滑、编造倾向、审计链断裂、多 agent 失配、工具越权。每一类命门都不该只停在名称上，而应化成可重复的攻击面与试场。例如，要测试目标漂移，就不能只问一轮短题，而要放进长链任务、插入中途诱因，看系统是否换题而不自知；要测试记忆污染，就要设计错误材料如何经客念、气海、经脉一路潜入主线；要测多 agent 失配，就要制造角色信息不对称与权限冲突，观察系统是否能裁决。

真正的高阶评测，还必须能回流。若独孤镜照出了某条命门，结果不能只停在报告里，而应回到原则层调整规则、回到省察层优化告警、回到知行环补工件、回到传习阁写教训、回到医毒堂完善防线。评测若不能改系统，就只是围观；能改系统，才算破招。

故本卷之意，是把“测得像对”换成“测得见骨”。系统不是因听过夸奖而强，乃是因见过自己的死穴、练过自己的弱点、留下了破招之法，方能真正久行。
