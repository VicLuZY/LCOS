# 卷十五 传习阁：经验蒸馏与传承

### 小节一代际非重装：须可传

若每一代系统皆从头学，则非 OS，乃不断重装。**传习阁**司：成功经验入谱，失败教训入狱，招式蒸馏成法，门规随实践更新，版本变更可追溯。此乃组织之学，非单次任务之学。

### 小节二蒸馏之三昧：去冗、留骨、可执行

蒸馏须去冗、留骨、可执行：从案例中抽出可复用规则与反例，而非堆叙事。规则须与本体层对齐，反例须与破贼录对齐。否则谱成杂物房。

### 小节三传承与偏院：新法如何入谱

新法自偏院出，须经试法记录、镜检标准、评测结果，方可入谱。入谱须变更编号、影响面说明、回滚指引。无回滚指引者，不得谓可传承。

### 小节四工程落点：知识库 schema 与签章

工程落点：**知识库 schema**（规则、反例、适用范围、置信度）、**数字签章或审批记录**（谁批准入谱）、**只读镜像**（生产读谱，写谱在阁）。无 schema 则检索不可依赖；无签章则责任不可追。
