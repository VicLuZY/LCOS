# 卷十三 阵法与书院：多 Agent 编排

众器不怕多。
怕无主。

单一系统之乱，尚可追线。
多主体之乱，一旦并起，最容易变成众口有词，竟无人负责。

所以龙场OS讲阵法。
不是为了把编排说得热闹。
是为了把责权说得清楚。

多 Agent 不是多几个会说话的分身。
而是把不同职责，分给不同位置。

有人掌门。
总揽目标、边界、取舍与最后裁断。

有人游侠。
宽搜外物，四面探路。

有人长老。
持经据典，守主脉与旧录。

有人护法。
真正下手执行，守工具与门闩。

有人记室。
只记已发生之事，不替任何人添彩。

有人弟子。
承小任，做局部，逐层升阶。

角色一清，才谈协议。

谁在说。
为何而说。
凭什么说。
此话与哪条任务、哪份证据、哪一版记忆相关。

这些若不明。
并行只会把混乱放大。

所以阵法还要管可见域。

不是每个角色都该看见同样的东西。
不是每个角色都能写同样的地方。
私池、别院、主脉、冷渊，要各守其门。

冲突也不必怕。
多主体之中，有争很正常。
可怕的是争而无裁。

事实冲突，拿证据裁。
目标冲突，拿本体层裁。
权限冲突，拿门闩裁。
仍裁不下，再请人来定。

阵法若成，多器可以互补。
阵法若坏，多器只会互污。

所以本卷又讲书院。

因为多主体不是天然会协作。
要教。
要练。
要让新人先在沙池走，再入别院，再近主脉。

经验若只散在聊天里。
下一批角色仍会从头犯旧错。

把裁决写成教材。
把失败写成判例。
把升降写成门规。

这才叫书院。

阵法以理，书院以续。

没有阵法，众器只成众噪。
没有书院，阵法终会失传。
